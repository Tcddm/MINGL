# 填充多边形
填充多边形采用**标准扫描线填充算法**
::: tip 相关配置
MGL_FILL_POLYGON_MAX_INTERSECTS
:::

## 使用场景
用于绘制任意形状的实心多边形

## 算法原理
- **逐行填充**：从上到下逐行扫描，计算每行与多边形的交点区间，再逐个填充。适合点阵屏的离散像素特性。
- **只计算交点**：不需要对每个像素做是否在多边形内的判断，计算量小，在MCU上也能流畅运行。
- **自动处理凹凸多边形**：算法天然支持任意凹凸多边形，只要不自相交。
- **闭合多边形自动填充**：首尾顶点自动连接，无需手动闭合。

## 绘制函数

::: tip 依赖的绘制函数

:::
::: warning
仅支持无自相交、无内孔洞的简单多边形，自相交多边形填充会错乱
:::
